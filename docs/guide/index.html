🚀 Start rapid (recomandat)

Scenariul standard (recomandat)

  1. Lasă Auto Plan (⏰) ON.
  2. Apasă ADD (📁) și importă videouri.
  3. Verifică / ajustează Targets (YT/IG/TT) și Visibility.
  4. Pe fiecare platformă (YouTube/Instagram/TikTok) apasă Schedule dacă vrei oră diferită.
  5. Activează Auto Upload (🚀) doar când ești gata.

Scenariul “manual control”

  1. Setează Auto Plan (⏰) OFF.
  2. Importă videourile (nu se programează automat).
  3. Activează target-ul dorit (ex: doar YT) și apasă Schedule pe coloana platformei.
  4. Pornește Auto Upload (🚀) doar când vrei execuție automată.
Important: “Planificarea” (Auto Plan / Plan dialog) este diferită de “execuție” (Auto Upload). Poți avea schedule setat fără să ruleze uploadul automat.

📁 Import videouri

ADD (📁) → Files / Folder

  • Files: selectezi unul sau mai multe fișiere video.
  • Folder: selectezi un folder; aplicația scanează și importă toate videourile suportate.
Auto Plan ON: după import, aplicația calculează automat date/ore și creează job-uri pentru platformele active.
Auto Plan OFF: după import, nu se face programare automată și targets sunt OFF (default).

🎯 Targets + Visibility

Targets (YT / IG / TT)

  • Apeși pe chip-ul YT / IG / TT ca să activezi sau dezactivezi.
  • Targets se salvează ca “preferințe pe rând” (persistă), fără să creeze job-uri până când programezi.

Visibility

  • Setează Private / Unlisted / Public (relevant pentru YouTube).
Screenshot 06 — Targets în tabel (chip-uri YT/IG/TT).

⏰ Schedule per platformă (YouTube / Instagram / TikTok)

Cum programezi doar o platformă

  1. Asigură-te că target-ul platformei e ON (ex: YT).
  2. În coloana platformei, apasă Schedule.
  3. Alege Later și setează Date & Time.
  4. Apasă Schedule → se creează job doar pentru acea platformă.
Per-platform: poți avea ore diferite pentru YT vs IG vs TT, pe același video.
Screenshot 07 — Dialog “Schedule for …” (Now / Later + Date & Time).

📅 Plan (bulk planificare pe selecție)

Când folosești Plan

  • Când vrei să aplici același “pattern” de ore pentru mai multe rânduri selectate.
  • Poți alege “Apply to: YouTube / Instagram / TikTok / All active”.

Pași

  1. Selectează rândurile (checkbox).
  2. Apasă Plan.
  3. Alege start date, ore, timezone și “Apply to …”.
  4. Apasă Apply.
Screenshot 08 — Dialog “Plan” (bulk schedule).

🚀 Publish (workflow de execuție)

Publish este un flow “operational”: poți genera metadata și/sau porni upload/schedule. Recomandare: stabilește întâi Targets + Schedule, apoi folosește Publish/Auto Upload când ești gata.

🤖 Auto Upload (execuție la timp)

Ce face Auto Upload

  • Verifică periodic job-urile programate (“due”).
  • YouTube: upload automat.
  • Instagram/TikTok: Manual Assist (te ajută, nu postează automat).

Silent mode

  • ON: doar notificare; la click se copiază text + se deschide browser/Explorer.
  • OFF: copiază text + deschide browser/Explorer imediat (o singură dată, nu repetă la fiecare poll).
Advanced — de aici pornești Auto Upload + Silent Mode.
Important: aplicația trebuie să fie pornită (în tray) ca Auto Upload să ruleze.

🧯 Probleme frecvente

Port 5173 ocupat

Dacă npm run dev spune “Port 5173 is already in use”, închide procesul care ține portul:

netstat -ano | findstr :5173
taskkill /PID <PID> /F

Auto Plan OFF dar vrei să programezi

  • Activează target-ul (ex: YT) și apasă Schedule în coloana platformei.

YouTube nu se conectează

  • Advanced → “Test connection” / “Reconnect”.
  • Asigură-te că ai google_oauth_client.json în userData.
